# draw.io Gemini Extension

This extension allows Gemini CLI to generate and open diagrams directly in the draw.io editor.

## Available Tools

- `open_drawio_xml`: Opens a diagram from XML content.
- `open_drawio_csv`: Opens a diagram from CSV content.
- `open_drawio_mermaid`: Opens a diagram from Mermaid.js syntax.

## Features

- Supports Flowcharts, Sequence Diagrams, Class Diagrams, and more via Mermaid.
- Supports native draw.io XML for complex diagrams.
- Supports CSV import for data-driven diagrams like Org Charts.
